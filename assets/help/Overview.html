General Overview

ServeStream was designed for users who want to browse and play streams from one or more media servers. It supports playback of the following android supported media formats with additional support for playlist (m3u) files:

Audio: 3gp/mp4/m4a/mp3/ogg/wav/mid/xmf/mxmf/rtttl/rtx/ota/imy
Video: 3gp/mp4

ServeStream offers the ability to browse media servers that serve HTML pages. This is accomplished by parsing links from the HTML that is returned when performing a HTTP GET request against a specified URL. The ability to browse a media server allows users to access files without having to type or remember fully qualified URLs.
